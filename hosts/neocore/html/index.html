Services on AfRA core:

presence monitor

Announce your presence at AfRA, by detecting when your devices are on the network.

mpd

Music player daemon. Zeige Deinen Lieblings-MPD-Client auf core.lan Port 6600 (Standard-Port)

ympd

Web client für mpd

pulseaudio

PulseAudio lauscht auf dem Standard-Port 4713.

export PULSE_SERVER=core.lan

In der gleichen shell gestartete anwendungen streamen den output an core

oder:

pacmd load-module module-tunnel-sink-new server=core.lan

oder für pipewire:

pactl load-module module-tunnel-sink server=core.lan

Ein neues output device ist auf dem lokalen pulseaudio-server verfügbar

snapcast

Synchronisierte Audiowiedergabe auf alle Musiksysteme. Wird von pulseaudio und mpd als standard-output verwendet.

snapweb

Web client zum Anpassen der Lautstärke und Latenzanpassung von snapcast

nfs

Befolge die Anweisungen Deiner Distribution zur Einrichtung als NFS-Client.

Z.B. für Arch Linux: https://wiki.archlinux.org/title/NFS

sudo mount -t nfs -o vers=4 core.lan:/ /mnt/core

Minetest

Wir betreiben einen Minetest (Minetest Game) server auf afra-berlin.eu:30000!

Beim initialen connect kann es sein, dass man kein passwort setzen kann. Das kann man allerdings im Spiel auch nachholen: ESC -> Change Password
